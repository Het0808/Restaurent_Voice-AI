# Multilingual AI Restaurant Voice Receptionist

A portfolio-grade AI voice receptionist designed to receive restaurant phone calls and assist guests naturally in English, Hindi, and Gujarati.

The planned system will support concise spoken conversations, restaurant information retrieval, reservation workflows, interruption handling, and human handoff. It will keep live reservation availability in the transactional database while using retrieval-augmented generation only for restaurant documents.

## Project status

Stage 0, project initialization, is complete. Application code has not been created yet. See [project context](docs/project-context.md) for the objective and planned stack, and [progress](docs/progress.md) for the staged roadmap.
